The 404 error on /auth is because Appwrite environment variables are not set.

1. Copy the environment template:
   cp .env.template .env

2. Edit .env with your Appwrite credentials:
   - Go to https://cloud.appwrite.io
   - Select your project  
   - Go to Settings â†’ General
   - Copy your Project ID
   - Replace 'your-project-id-here' in .env

3. Restart your development server:
   npm run dev

4. Test Google authentication:
   - Visit http://localhost:3000/get-started
   - Click 'Continue with Google'

The error message will now be more helpful if configuration is missing.
